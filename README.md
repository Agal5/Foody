# 🍔 Foody - Food Ordering UI Website

Foody is a modern and responsive food ordering website UI built using HTML and CSS.  
It showcases a clean layout with visually appealing food cards and a "Popular Right Now" section inspired by real-world food delivery apps.

---

## ✨ Features

- 🍕 Clean and modern UI design  
- 📱 Responsive layout using Flexbox  
- 🧾 Food cards with image, name, price, and time  
- 🎯 Focus on spacing, alignment, and typography  
- 🎨 Hover effects for better user experience  

---

## 🛠️ Technologies Used

- HTML5  
- CSS3
